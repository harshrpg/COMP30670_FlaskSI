============
Installation
============


Stable release
--------------

To install Flask System Info, run this command in your terminal:

.. code-block:: console

    $ pip install flask_system_info

This is the preferred method to install Flask System Info, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Flask System Info can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/harshrpg/flask_system_info

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/harshrpg/flask_system_info/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/harshrpg/flask_system_info
.. _tarball: https://github.com/harshrpg/flask_system_info/tarball/master
